挂载外置设备支持

可挂载500G以下的外置USB硬盘盒，系统可自动识别。

系统添加了自动挂载的脚本，插入U盘即可自动挂载为/mnt/sda1，插入TF卡，则自动挂载为/mnt/mmcblk1p1。目前尚未测试多分区是否自动挂载，熟练者可以自行尝试。支持文件系统的格式为NTFS和EXT4.

强烈建议将U盘格式化为ext4文件系统:
  • * 部分盒子建议插到靠近电源的USB口。
  • * 建议使用linux自带的fdisk分区命令进行重新分区。
  • * 使用mkfs.ext4 /mnt/sda1命令格式化。
外置大硬盘盒可能需要注意供电是否正常:
  • * 尝试重新插拨，查看是否自动挂载。
  • * 部分机型可能需要插上USB口以后热启动（上电）才会被盒子识别。
格式化USB外置储存设备指南 
  • 			    		umount /dev/sda1		先卸载
			    		fdisk -l	查看磁盘情况
			    		fdisk /dev/sda	进入命令行分区界面
			    		（注意，这里是sda而不是sda1）
			    			输入 o 设置新分区为msdos格式
			    			输入 n 建立新分区
			    			接着输入 p 建立主分区
			    			接着默认2048字节扇区头和自动的扇区尾
			    			(按回车即默认)
			    			输入 w 将上面的操作保存。
			    		mkfs.ext4 /dev/sda1	格式化它
			    		TF卡请自行更换为mmcblk1(对应sda)
			    		mmcblk1p1(对应sda1)
			    		mkdir -p /mnt/sda1/xxx	建立文件夹xxx
			    		chmod 777 -R /mnt/sda1/xxx 赋予全部权限

                                    	ExecStart=/usr/bin/dockerd --graph="/mnt/sda1/dockerfiles" -H fd:// --containerd=/run/containerd/containerd.sock
                                    	systemctl daemon-reload
                                    	systemctl start docker
